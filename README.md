# RegistroDeEstudiantes-Manejo-de-Excepciones
Descripción: Este es un programa realizado con Windows Forms que trata sobre el registro de estudiantes con Nombre, edad, correo y teléfono. En este se hizo el uso del try-catch para hacer el manejo de excepciones que se pudiesen presentar dentro del programa.

Herramientas utilizadas:

Lenguaje: C#
Framework: .NET Framework
IDE: Visual Studio 2022
